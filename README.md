MSME Scheme Recommender Systems

This repository contains a Backend (Node/Express) and Frontend (React) for the MSME scheme recommender project.

Remote: https://github.com/bharathganesh45/MSME-scheme-recommender-systems.git

How to push local workspace to the remote (run from workspace root):

```bash
# initialize local git (if needed)
git init
git checkout -b main

# stage and commit
git add .
git commit -m "Initial commit"

# add remote (use HTTPS or SSH as you prefer)
git remote add origin https://github.com/bharathganesh45/MSME-scheme-recommender-systems.git

# push
git push -u origin main
```

If the remote doesn't exist, create it using the GitHub CLI:

```bash
# login and create repo
gh auth login
gh repo create bharathganesh45/MSME-scheme-recommender-systems --public --source=. --remote=origin --push
```
